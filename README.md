# trip-planner
ICHack2017 Trip Planning Web Application


###Run Locally:

Turning on debug mode. In Command Line in project root folder:

    git clone https://github.com/terimater/trip-planner
    cd trip-planner
    set FLASK_CONFIG "development"
    python runserver.py
